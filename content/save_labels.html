Save Labels

Save Labels#

In the previous step, we delinated zone and their associated labels. Those label need to be save using the ‘Save Labels’ button.

Note

You can manually edit those label by openning the label file and adding/removing labels. Those will then appear inside the GUI.

Warning

If you manually edit the label file, do not remove the first 2 lines ‘__ignore__’ and ‘_background_’
