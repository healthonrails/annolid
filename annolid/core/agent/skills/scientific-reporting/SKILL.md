# Scientific Reporting

Use this skill when the user wants an interpretable analysis summary rather than
raw artifacts.

## Structure

- assay/task context,
- evidence and methods used,
- key findings,
- limitations or ambiguity,
- pointers to reproducible artifacts.

## Rules

1. Ground every claim in available evidence or stored outputs.
2. Keep tone scientific and concise.
3. Separate what the data shows from what it may imply.
4. Do not bury caveats.
