# Experimental Design

Use this skill when the bot needs to connect behavior outputs to a meaningful
experimental question.

## Focus

- identify the assay objective,
- define the metric that answers it,
- preserve comparability across runs,
- call out missing controls or ambiguous setup.

## Guidelines

1. Prefer measurable endpoints over narrative summaries.
2. Keep outputs compatible with replayable artifacts and validation.
3. If the user asks for a comparison, state the denominator and time window.
4. If the assay framing is underspecified, choose a conservative metric set and
   note what is missing.
