# Annolid Docs

Annolid is a practical toolkit for annotation, segmentation, tracking, and behavior analysis in real lab workflows.

This portal is the documentation entry point for installation, day-to-day usage, deployment, and reference material.

## Start Here

- [Installation](installation.md): Pick a supported install path for your system.
- [Workflows](workflows.md): Run the standard video-first labeling and tracking workflow.
- [Tutorials](tutorials.md): Follow curated walkthroughs and practical guides.
- [Reference](reference.md): Jump directly into operational and technical documentation.
- [Deployment](deployment.md): Understand how the landing page, docs, and book are published.

## Documentation Surfaces

- `website/`: source of truth for the root landing page at `annolid.com/`
- `docs/`: source of truth for the MkDocs docs site and `/portal/`
- `book/`: Jupyter Book content published at `/book`
