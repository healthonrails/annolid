# Get in touch

## Documentation and community
- Documentation site: https://annolid.com (mirror: https://cplab.science/annolid)
- GitHub repository: https://github.com/healthonrails/annolid
- Issues / bug reports: https://github.com/healthonrails/annolid/issues
- YouTube channel: https://www.youtube.com/@annolid/videos
- Annolid Google Group: https://groups.google.com/g/annolid

## What to include in a bug report
When possible, include:
- Your OS and Python version
- Annolid version (`annolid --version`)
- A minimal example (video snippet, a few JSONs, or a small dataset export)
- The relevant logs / error traceback
