# Video Depth Anything (Depth Estimation)

Annolid bundles the [Video-Depth-Anything](https://github.com/DepthAnything/Video-Depth-Anything) model family so you can estimate per-frame depth for a video directly inside the GUI (or from Python).

For the maintained step-by-step guide (GUI flow, checkpoint handling, Python API examples, and output formats), see:

- [Video Depth Anything (Docs)](https://annolid.com/video_depth_anything.html)

Quick reminder:

1. Open a video in Annolid.
2. (Optional) tune settings via **View → Depth Settings…**.
3. Run **View → Video Depth Anything…**.
